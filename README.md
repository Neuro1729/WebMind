# WebMind
A mini Perplexity-style agent that searches the web and answers with sources.

A small AI agent that:

Detects if a question needs web search

Calls Tavily Search API

Fetches top webpages (requests + Newspaper3k / BeautifulSoup)

Extracts text

Summarizes + answers using Qwen 2.5B / Llama 3 3B

Returns clean answer with citations
